# WaterIntelligence
WaterIntelligence is a simple extension that shows your water consumption by using AI. 
The system is based on the characters used; statistics on water consumption may vary depending on the servers, models, files passed as parameters, and other values. These are average values and may be inaccurate.

# How to install it
1. Download the project with Code > Download zip or ```git clone https://github.com/Baron-Komodo/WaterIntelligence.git```
2. Open your chrome browser, and type chrome://extensions in the url
3. Enable Developper mode
4. Then click on Load unpacked and choose the folder of this project
